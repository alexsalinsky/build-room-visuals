The simple path

Cloudflare is the cleanest, fastest, free way to get your robot's work online.

1
Sign up for a Cloudflare account Free tier. Email and password. That's it.
2
Create an API key Inside Cloudflare, under My Profile, API Tokens.
3
Give the API key to your robot Codex, Claude Code, whatever you're using. Drop it in your .env file.
4
Tell your robot to publish to Cloudflare using the credentials Your robot handles the rest. You get a live URL.
Bonus

Cloudflare has its own LLM service built in (Workers AI). Your robot can call AI directly through Cloudflare.

No separate API token hookup. No new payment method.
