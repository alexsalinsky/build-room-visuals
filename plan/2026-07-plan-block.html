HOW TO USE THIS
Open a block. Pick 1-6 tasks from the theme's rolling to-do list (Altitude 2). Sum to ~90 min. Write one sentence for "done." Execute. Cross off. If you don't finish, tasks roll to the next block of that theme, not to another theme.

Rules for filling a block

  • Bucket blocks hold 3-6 tasks from one theme's rolling list. Sum to ~90 min.
  • Focus blocks hold 1 task. When a single thing earns the whole 90 min, give it the whole 90 min.
  • Every task carries a duration estimate (5 / 10 / 15 / 30 / 45 / 60 min) and a one-sentence outcome.
  • "Done" is written before starting. If it's fuzzy, the block is fuzzy.
  • Blocked tasks stay in the rolling list, not the block. Add them when their dependency clears.
  • Undone tasks roll to the next block of the same theme, not another theme.
  • Grab-bag is the only theme where miscellaneous belongs.
