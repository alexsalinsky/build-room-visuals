Install Codex.

Lowest-friction install for people who aren't already in an AI coding tool. Runs in your terminal, signs in with your ChatGPT account, works on Mac, Linux, and Windows.

  1. Open your terminal.
  2. Paste and run the install command for your OS:
    # Mac or Linux curl -fsSL https://chatgpt.com/codex/install.sh | sh # Windows (PowerShell) powershell -ExecutionPolicy ByPass -c "irm https://chatgpt.com/codex/install.ps1 | iex"
  3. Run codex and sign in with your ChatGPT account.

Official install page: github.com/openai/codex. If curl fails on Mac, brew install --cask codex also works.
