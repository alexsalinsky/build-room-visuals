Same brain, different handsClaude chat. Cowork. Claude Code. Codex. What's the actual difference?

You keep hearing four product names and trying to pick the "best" one. That's the wrong question. They're the same model wearing different gloves. The real question is:

What surface is the AI allowed to act on?

Three tiers, increasing capability. Pick the lowest tier that touches what you actually need to change.

TIER 1

Talk

AI you use
What it seesOnly what you paste into the chat
What it can changeNothing on your computer
Where output livesIn the conversation. You copy-paste to apply.
Examples: claude.ai chat, ChatGPT, Gemini chat, Perplexity
TIER 2

Touch your files

AI that edits
What it seesA folder or doc you point it at
What it can changeReads and writes files in that surface
Where output livesFiles on your disk or in your Drive update directly
Examples: Claude Desktop with Cowork mode, ChatGPT with file access, Notebook-style docs
TIER 3

Build and run

AI you build with
What it seesA whole project, your terminal, the internet
What it can changeEdits files, runs commands, installs packages, starts servers
Where output livesA working product that exists outside the chat
Examples: Claude Code (CLI), Codex (CLI), Cursor, agent frameworks

Which tier should I open right now?

I want to think out loud, brainstorm, or learn Tier 1 (chat). Pasting back and forth is fine here.
I want a real document or set of docs updated Tier 2 (Cowork or equivalent). Point it at the folder.
I want a tool that runs by itself when I'm not there Tier 3 (Claude Code or Codex). Live in the terminal.

How this lines up with "inside the robot"

From S1, the AI build stack:

chatbox → LLM → context → integrations → product

Tier 1 lights up the first two boxes. You give context by typing.

chatbox → LLM → context → integrations → product

Tier 2 adds context. The AI reads your files and writes them back.

chatbox → LLM → context → integrations → product

Tier 3 lights the whole chain. Now there's a product that exists outside the system.

What changes between tiers

  • The model is the same. Claude Sonnet 4.X powers claude.ai, Cowork, and Claude Code. The "intelligence" doesn't get smarter at tier 3.
  • The surface gets bigger. Tier 1 sees one message. Tier 2 sees a folder. Tier 3 sees your whole machine and the internet.
  • The risk gets bigger. Tier 1 can waste your time. Tier 3 can delete a file. Use planning mode and confirm before destructive moves.
  • The cost gets bigger. Same-priced per-token, but tier 3 reads more context per task. Budget accordingly.

Two things you do NOT need to pick between

  • Claude vs ChatGPT vs Gemini. Same tier, slightly different models. Pick one, get fluent. Switching later is cheap.
  • Claude Code vs Codex. Same tier (3). Same job. Try both for a week, keep the one whose vibe you like.

What you DO need to pick between

Tier 1 vs Tier 2 vs Tier 3 for the thing in front of you right now. That's the whole question.

If the answer is "I want it to make a thing that keeps running" — you're at Tier 3. Open the terminal.
